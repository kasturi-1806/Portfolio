# Personal Portfolio Website

This is my personal portfolio website built using **HTML and CSS**. The website is designed to showcase my skills, projects, and contact information in a clean and responsive layout.

## Features

* Responsive navigation bar
* Hero section with introduction
* About section
* Skills section with progress bars
* Projects showcase
* Contact form
* Footer with social links

## Technologies Used

* HTML5
* CSS3
* Font Awesome (for icons)

## Project Structure

* `index.html` – Main structure of the website
* `style.css` – Styling and layout
* `images/` – Images and assets used in the website

## Purpose

I created this project to practice frontend development, improve my understanding of **responsive design**, and build a portfolio that represents my work as a Computer Engineering student and aspiring Frontend Developer.
